# OSG_SermonScript (열린하늘문교회 설교문)

## 파일 설명
열린하늘문교회 주일예배 및 수요예배 등 설교를 텍스트로 추출한 파일을 게시

## 폴더 설명
* SUN: 주일예배 설교
* WED: 수요예배 설교

## 설교 요약
### 수요 예배
#### 2024/05/22 김동현 목사 (여전히 우리를)
* 나오미의 삶이 하나님의 은혜로 회복되었습니다.
* 다른 사람의 은혜와 선행을 통해 일어나는 작은 변화들이 하나님의 일하심입니다.
* 고난과 문제 앞에서 큰 변화를 기대하기보다는 작은 변화에서 하나님의 손길을 느낄 수 있습니다.
* 하나님의 은혜와 사랑은 변함없이 우리의 삶을 추격해 오며, 우리를 회복시키십니다.
* 우리는 서로에게 하나님의 은혜를 흘려보내는 통로가 되어야 합니다.
* 하나님은 우리의 삶을 주목하시고 개입하시는 분이시며, 우리는 그 하나님께 소망을 두어야 합니다.
